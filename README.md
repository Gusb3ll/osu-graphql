# osu-graphql

Graphql server for osu!

## Setup

Install dependencies

```bash
yarn install
```

## Development

```bash
yarn dev
```

access apollo server at localhost:3333/graphql

## Production

```bash
yarn build

yarn start
```

apollo server will be running at localhost:3333
